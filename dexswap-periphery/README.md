# DexSwap Periphery Contracts

## Local Development

The following assumes the use of `node@>=10`.

## Clone Repository

`git clone https://github.com/Agin-DropDisco/dexswap-periphery.git`

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`

## Flatten Contracts

`yarn flattener`
